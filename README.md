# GSAP Front Gate Tickets Style Menu

A clean, conflict-resistant recreation of the mobile navigation + search menu animation used on [Front Gate Tickets](https://www.frontgatetickets.com).

## Features
- Matches the original hamburger line compression animation
- Smooth staggered content reveal
- Reliable body scroll locking (`no-scroll`)
- Prevents GSAP conflicts (common when mixing Webflow + custom code)
- Full keyboard support (ESC key + focus management)
- Mobile-first with clean desktop fallback

## Installation

1. Copy js/menu-animations.js into your project
2. Include GSAP (recommended latest version):

```html
